# DID-007: Build default Connect Wallet UI

Labels: `type:feature`, `area:ui`, `priority:p0`, `slice:afk`

Milestone: v0.1

## What to build

Build the default UI package with a Connect Wallet button, chain-grouped wallet modal, connected account display, disconnect action, dark/light theme support, and responsive layout.

## Acceptance criteria

- [ ] Connect button reflects disconnected, connecting, connected, signing, and signed-in states.
- [ ] Wallet modal groups wallets by chain and shows install/connection status.
- [ ] Account display shows shortened address, chain label, and disconnect action.
- [ ] UI supports dark and light themes.
- [ ] Mobile layout is usable without text overflow.

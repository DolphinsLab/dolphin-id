# DID-012: Add WalletConnect v2 and mobile deep link login

Labels: `type:feature`, `area:adapter`, `priority:p1`, `slice:afk`

Milestone: v0.2

## What to build

Add WalletConnect v2 support for EVM wallets and mobile deep link flows so users can complete login from mobile wallet apps.

## Acceptance criteria

- [ ] WalletConnect v2 can connect an EVM wallet.
- [ ] Mobile browsers can launch supported wallet apps through deep links.
- [ ] Connection state is restored after returning from a wallet app.
- [ ] Failure cases have recoverable errors.
- [ ] Example app documents mobile setup and test steps.

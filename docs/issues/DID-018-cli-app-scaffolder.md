# DID-018: Build CLI app scaffolder

Labels: `type:feature`, `area:dx`, `priority:p1`, `slice:afk`

Milestone: v1.0

## What to build

Build a CLI scaffolder that creates a working Dolphin ID example app with selected chains, framework, token storage mode, and optional hosted service configuration.

## Acceptance criteria

- [ ] CLI can create a Next.js app with EVM and Sui login.
- [ ] CLI can select headless or default UI integration.
- [ ] CLI can select self-hosted or hosted nonce/session mode.
- [ ] Generated app runs tests and starts locally.
- [ ] CLI docs include common recipes.

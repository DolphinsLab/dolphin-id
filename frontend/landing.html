AUTH FOR MULTI-CHAIN APPS

One login layer for EVM, Sui, Solana, Bitcoin, Aptos.

Dolphin ID gives dApps a self-hosted SDK and an optional hosted nonce/session service, so teams can verify wallets without rebuilding auth plumbing for every chain.

SURFACES

Everything a developer needs to trust the path.

Self-hosted or hosted

Use the server package inside your own backend, or let Hosted Dashboard issue nonces and manage sessions.

Adapter-first chains

Wallet discovery, connection, signing, events and account selection stay behind a ChainAdapter contract.

Inspectable auth

Playground and debug tools expose SIWX messages, session state, wallet events and common error paths.

DEVELOPER FLOW

03

QUICK START

Install, configure, verify.

commands
$pnpm add @dolphin-id/react @dolphin-id/ui
$pnpm add @dolphin-id/server @dolphin-id/hosted
$dolphin-id create --framework next --chains evm,sui --auth hosted
01

Choose adapters for the chains your dApp supports.

02

Wrap the app in DolphinProvider and mount the UI package components.

03

Expose /auth/nonce, /auth/verify, /auth/session and /auth/logout on the server.

REACT ENTRY

import{ DolphinProvider, ConnectButton } from "@dolphin-id/react";
use<DolphinProvider projectId="proj_..." authMode="hosted">
